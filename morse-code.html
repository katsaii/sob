Notes

  • a dit (written as a dot: '.') has a signal length of 1, a dah (written as a dash: '-') has a signal length of 3
  • abnormal signal lengths, like those found in the american standard, are written directly as a plain hexadecimal digit
  • typically all non-word, non-letter pauses last a single bit, but in some cases a signal length of 0 indicates an extended pause
